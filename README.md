🐦 Game Hi Sahi

Ek immersive aur modern Flappy Bird Clone jo "Garden" theme par based hai. Ise pure JavaScript aur HTML5 Canvas ke saath banaya gaya hai, jisme natural physics aur real-time procedural audio ka istemal kiya gaya hai.

🚀 Live Demo

Aap is game ko apne browser mein direct chala sakte hain. Koi external plugins ki zaroorat nahi hai.

🎮 Game Controls

Jump: Spacebar, Arrow Up, ya Screen par Tap karein.

Restart: Game over hone ke baad screen par kahin bhi Tap karein.

Objective: Green fences (pipes) ke beech se bird ko nikalna hai bina takraye.

✨ Key Highlights

🛠 Technical Excellence

Frame-Independent Movement: requestAnimationFrame ka use karke 60FPS par smooth gameplay ensure kiya gaya hai.

Procedural Audio Engine: Game mein koi MP3 file nahi hai. Sabhi sounds (Jump, Score, Death, BGM) Web Audio API ke zariye code se real-time generate hote hain.

Responsive Scaling: Portrait mode (400x600) ke liye optimized layout.

🎨 Visual Aesthetics

Parallax Background: Garden clouds aur layers alag-alag speed par move hote hain.

Animated Water: Bottom mein waves ka dynamic effect.

Natural Bird Rotation: Bird ki speed ke hisaab se uska angle change hota hai (Smooth tilt-up/down).
